# 功能
### 導航欄
<img src="https://upload.cc/i1/2022/06/17/Iq2GUu.png" width=70%>

* Home 就是一開始2048的首頁
* Notes 點下去會連結到期末筆記的部分
* Mail 點下去會連結到信箱寄信給我
* Portfolio 點下去會連結到之前做的相關作品
* Me 則是會連接到我的個人網頁

### Home
<img src="https://upload.cc/i1/2022/06/17/CazksF.png" width=50%>

* 這個遊戲玩法是利用鍵盤移動，使兩個相同的數字碰撞再一起並合成更大的數字(ex: 兩個數字為2的方塊碰撞在一起後可以合成出4，兩個4可以合出8，以此類推)
* 遊戲的勝利目標是合成出2048這個數字
* 如果方塊多到塞滿了16格就輸了
* 可以使用上下左右鍵開始遊戲
* 輸了可以點擊旁邊的重新開始
* 左下角為分數計算，重新開始後也會歸0

### Notes
<img src="https://upload.cc/i1/2022/06/17/Yvrmng.png" width=50%>

* 左方欄位可以點擊連結至其他筆記
* 其他筆記下方也有書籤至各個標題
* 筆記最下方都有附上圖片使用來源以及參考資料

### Mail
<img src="https://upload.cc/i1/2022/06/17/0TWGsc.png" width=50%>

* 點擊後會打開信箱並郵寄給我

### Portfolio
<img src="https://upload.cc/i1/2022/06/17/KhPT5y.png" width=50%>

* 左邊其他作品欄位欄位可連結至其他的的作業作品

### Me
<img src="https://upload.cc/i1/2022/06/17/HOrK1X.png" width=50%>

* 這裡連結至我的個人網頁
* 右下手機上的app皆可點擊連結至我的社群

# 專案內容

### 導航欄

* 重新編排了導航欄的位置
* 新增導航欄連結至我的各個網頁
* 非原創，大約修改了15%

### 2048(Home)

* 改變2048的整體比例，使遊玩順暢 (原比例過大，導致使用上下鍵時，網頁會一直跟著上下移動)
* 重新編排位置，並新增了重新開始的功能按鈕 
* 非原創，大約修改了15%

### 筆記(Notes)

* 筆記皆是自行攥寫，除了使用圖片外，無使用任何模板及外部資源
* 原創

### 作品集(Portfolio)

* 皆是練習作業時自行完成的，無使用任何外部資源
* 原創

### 個人網頁(Me)

* 除了使用圖片外，排版與設計皆是原創，無使用任何模板
* 原創
